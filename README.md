## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Vignesh061/HomyStay.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Guest-House-Booking-System-Html-CSS-JavaScript
   ```
3. Open `index.html` in your browser.
